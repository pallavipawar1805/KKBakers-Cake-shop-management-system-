# Bakery Management SQL Commands Demo

## Overview
This project demonstrates **key SQL commands and concepts** using a **sample bakery management database**. It is designed as a **college-level educational project** to showcase practical SQL usage.  

The project covers database creation, table management, data manipulation, querying, joins, subqueries, window functions, transactions, and advanced SQL operations.

---

## Database Schema
- **Database Name:** `KKbakers`  
- **Tables:**  
  - `department` – Bakery departments  
  - `staff` – Staff members with department info  
  - `suppliers` – Supplier details  
  - `mk_product` – Raw materials and supplied products  
  - `my_product` – Bakery products  
  - `customers` – Customer information  
  - `orders` – Customer orders  

---

## SQL Concepts Demonstrated
- **DDL:** `CREATE`, `ALTER`, `DROP`, `RENAME`  
- **DML:** `INSERT`, `UPDATE`, `DELETE`  
- **Querying:** `SELECT`, `WHERE`, `LIKE`, `BETWEEN`, `DISTINCT`  
- **Joins:** `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `CROSS JOIN`, `UNION`, `UNION ALL`  
- **Subqueries:** Scalar, multiple-row, correlated, inline views  
- **Window Functions:** `ROW_NUMBER`, `RANK`, `DENSE_RANK`, `LEAD`, `LAG`, `NTILE`  
- **Aggregate Functions:** `SUM`, `AVG`, `COUNT`, `MAX`, `MIN`  
- **Transactions:** `START TRANSACTION`, `COMMIT`, `ROLLBACK`  
- **String Functions:** `CONCAT`, `SUBSTRING`, `UPPER`, `LOWER`, `TRIM`, `REPLACE`  
- **Mathematical Functions:** `ROUND`, `CEIL`, `FLOOR`, `ABS`, `POWER`, `SQRT`, `MOD`, `SIGN`, `TRUNCATE`, `RAND`  

---

## Features
- Demonstrates **table relationships** using foreign keys and cascading actions.  
- Includes **sample data** for staff, customers, products, suppliers, and orders.  
- Example queries for:  
  - Top-selling products  
  - Best customers  
  - Revenue per product  
  - Customer order analysis  
- Repeated **data manipulation commands** for learning purposes.  

---

## How to Use
1. Clone or download the repository.  
2. Open the SQL file in **MySQL Workbench** or any SQL client.  
3. Execute commands **section by section** to see results.  
4. Explore **joins, subqueries, window functions, and aggregate queries**.
